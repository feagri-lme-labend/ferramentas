# Ferramentas Computacionais – LabEND

Esta seção reúne ferramentas desenvolvidas no laboratório para apoiar atividades de pesquisa, incluindo:

- processamento de dados experimentais
- organização de arquivos de ensaio
- modelagem e cálculo estrutural
- análise de sensibilidade
- apoio a ensaios não destrutivos

As aplicações podem ser utilizadas de duas formas:

- **Aplicações web** disponíveis no portal do laboratório
- **Scripts locais** disponíveis nos repositórios do GitHub

## Acesso

As ferramentas disponíveis na nuvem podem ser acessadas pelo portal:

Portal LabEND  
https://labend1.streamlit.app/links_auxiliares

Cada ferramenta possui um repositório próprio contendo:

- código-fonte
- documentação
- instruções de uso
